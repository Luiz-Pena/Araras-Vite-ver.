## Configurar variáveis
Em backend .env.example altere o DB_PASS pela senha atual do seu MySql
O arquivo deve parecer mais ou menos assim
    
    DB_HOST=localhost
    DB_USER=root
    DB_PASS=mysql
    DB_NAME=siteAraras

## Instalar e rodar
    
    1- cd backend 
    2- npm install 
    3- npm run dev
    (Abra outro terminal)

    4- cd frontend 
    5- npm install 
    6- npm run dev
